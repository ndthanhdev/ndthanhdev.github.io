# Finding Docs

Detailed procedure for locating relevant documentation before acting. Referenced
by the "Find Docs" section of `SKILL.md`.

## Steps

1. Read `skills/repo-docs/state/index.md` for an overview of what documentation
   exists across all categories.
2. Grep `docs/` for keywords (component names, feature names, patterns), and
   skim the category that fits your task — see the category table in `SKILL.md`.
3. Read the relevant docs and incorporate their guidance into your work.
4. If no docs cover your area, note the gap — you'll fill it during indexing.
